# tarcianne-code.github.io
Servidor/web site aberto a todos os públicos com o intuito de doar ideias, se inspirar em histórias, escrever e públicas suas próprias histórias. Inspired Tales surgiu de uma ideia em uma aula de Gerenciador de Conteúdo do curso de informática, com um desejo de tornar aquela ideia inspiradora em algo real para ajudar jovens escritores.

Server/website open to all audiences with the purpose of donating ideas, finding inspiration in stories, writing, and publishing your own narratives. Inspired Tales emerged from an idea in a Content Manager class during the computer science course, with the desire to turn that inspiring idea into something real to help young writers.
